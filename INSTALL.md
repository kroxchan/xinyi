# xinyi 安装指南

> 支持 macOS / Windows，下载 → 解压 → 双击即可运行。

---

## 下载安装包

进入 [Releases 页面](https://github.com/kroxchan/xinyi/releases) 下载对应版本：

| 操作系统 | 下载文件 |
|----------|----------|
| macOS (Apple 芯片 / Intel) | `xinyi-macos.zip` |
| Windows | `xinyi-windows.zip` |

---

## macOS 安装步骤

### 第一步：解压
双击 `xinyi-macos.zip` 解压，得到 `xinyi.app` 文件夹。

### 第二步：运行
双击 `xinyi.app` 即可启动。

> **首次打开提示"无法打开，因为来自身份不明的开发者"？**
> 系统设置 → 隐私与安全性 → 滚动到下方 → 点击"仍要打开"

### 第三步：打开 Dashboard
xinyi 启动后会自动打开浏览器，访问 `http://localhost:7872`
（如果没有自动打开，请手动在浏览器输入该地址）。

### 第四步：配置 API Key
在界面中找到 **设置 / Config** 标签页，填入：

| 字段 | 说明 |
|------|------|
| API Key | 你的 OpenAI / Anthropic / 兼容接口 Key |
| 模型 | 留空默认 `gpt-4o`，也可填 `claude-3-5-sonnet` 等 |
| Base URL | 使用 OpenAI 官方留空；硅基流动/火山引擎等填入代理地址 |

点击 **保存**，即可开始使用。

---

## Windows 安装步骤

### 第一步：解压
右键 `xinyi-windows.zip` → **解压到当前文件夹**（或"解压缩到 xinyi-windows"）。

### 第二步：运行
进入解压后的文件夹，双击 **`xinyi.exe`** 即可启动。

> 注意：会弹出一个黑色命令行窗口，显示运行日志，**请勿关闭它**。

### 第三步：打开 Dashboard
启动后会自动打开浏览器，访问 `http://localhost:7872`
（如果没有自动打开，请手动在浏览器输入该地址）。

### 第四步：配置 API Key
在界面中找到 **设置 / Config** 标签页，填入：

| 字段 | 说明 |
|------|------|
| API Key | 你的 OpenAI / Anthropic / 兼容接口 Key |
| 模型 | 留空默认 `gpt-4o`，也可填 `claude-3-5-sonnet` 等 |
| Base URL | 使用 OpenAI 官方留空；硅基流动/火山引擎等填入代理地址 |

点击 **保存**，即可开始使用。

## 文件说明

| 文件 | 作用 |
|------|------|
| `xinyi.app` / `xinyi.exe` | 主程序，双击运行 |
| `data/` | 聊天记录和训练数据（运行后自动生成） |
| `config.yaml` | 配置文件（首次保存后生成） |

## 常见问题

**Q: macOS 提示"无法打开"？**
> 系统设置 → 隐私与安全性 → 仍要打开

**Q: Windows 黑色窗口是什么？**
> 这是正常的日志窗口，显示程序运行状态，**请勿关闭**。关闭它会退出程序。

**Q: 首次运行很慢？**
> 首次会下载 AI 模型（约 500MB），耐心等待即可，后续无需重复下载。

**Q: 端口被占用？**
> xinyi 默认使用 `7872` 端口，如需修改请在 `src/app.py` 和 `src/__main__.py` 中查找 `server_port=7872` 并改为目标端口。

**Q: 浏览器显示"无法连接"？**
> 等待 10-20 秒，首次启动需要加载模型；如果超过 1 分钟仍然无法访问，检查黑色窗口是否有错误信息。

**Q: 想卸载？**
> 删除整个 xinyi 文件夹即可，无残留，无注册表。

**Q: 聊天记录存在哪里？**
> 本地 `data/` 目录，不会上传到任何服务器，完全私密。
